2. 관리자 페이지 접속 ✓
3. 이미지 업로드 및 콘텐츠 입력 ✓
4. 저장 ✓
5. 확인 ✓

```sql
TRUNCATE TABLE section_images;
```

## 2️⃣ 단계 2: 관리자 페이지 접속

### URL:
```
https://metamotion.io/admin.html
```

### 로그인:
- 이메일: (기존 관리자 이메일)
- 비밀번호: (기존 비밀번호)

---

## 3️⃣ 단계 3: Hero Slides 입력 (3개)

### 🔵 슬라이드 1

#### 이미지 업로드 (페이지 하단):

**한국어 이미지:**
1. 페이지 하단 "이미지 업로드" 섹션으로 스크롤
2. "파일 선택" 버튼 클릭 또는 드래그 앤 드롭
3. 한국어용 히어로 이미지 파일 선택 (예: hero1_kr.jpg)
4. "이미지 업로드" 버튼 클릭
5. **성공 메시지** 확인: "업로드 완료!"
6. 생성된 URL이 표시됨:
   ```
   https://metamotion.io/uploads/1765423456_abc123.jpg
   ```
7. **"URL 복사"** 버튼 클릭

**영어 이미지:**
1. 동일한 방법으로 영어용 히어로 이미지 파일 선택 (예: hero1_en.jpg)
2. 업로드
3. URL 복사

#### 콘텐츠 입력:

**한국어 섹션 (파란색):**
- **제목 (한국어)**: `실감형 VR 교육`
- **설명 (한국어)**: `몰입형 3D 레슨으로 완벽한 동작 학습`
- **이미지 URL (한국어)**: (위에서 복사한 한국어 이미지 URL 붙여넣기)

**영어 섹션 (초록색):**
- **제목 (영어)**: `Immersive VR Education`
- **설명 (영어)**: `Perfect motion learning with immersive 3D lessons`
- **이미지 URL (영어)**: (위에서 복사한 영어 이미지 URL 붙여넣기)

### 🔵 슬라이드 2

동일한 방법으로:
- 한국어 이미지 업로드 → URL 복사 → 붙여넣기
- 영어 이미지 업로드 → URL 복사 → 붙여넣기
- 제목, 설명 입력

**권장 내용:**
- 한국어: `국내 최대 모션캡처 스튜디오` / `OptiTrack Prime 41 카메라 48대 보유`
- 영어: `Korea's Largest Motion Capture Studio` / `Equipped with 48 OptiTrack Prime 41 cameras`

### 🔵 슬라이드 3

동일한 방법으로:
- 이미지 업로드 (한국어/영어)
- 제목, 설명 입력

**권장 내용:**
- 한국어: `AI 정밀 동작 분석` / `VRPE 기술로 실시간 자세 교정`
- 영어: `AI-Powered Motion Analysis` / `Real-time posture correction with VRPE technology`

### 💾 저장:
**"Hero Slides 저장"** 버튼 클릭 → "성공적으로 저장되었습니다!" 확인

---

## 4️⃣ 단계 4: Solution Slides 입력 (4개 권장)

동일한 방식으로:

### 슬라이드 1: K-POP 댄스
- 한국어 이미지 업로드 → URL 복사 → 붙여넣기
- 영어 이미지 업로드 → URL 복사 → 붙여넣기
- 제목/설명 입력

### 슬라이드 2: 태권도
### 슬라이드 3: 요가
### 슬라이드 4: 골프

**저장**: "Solution Slides 저장" 버튼 클릭

---

## 5️⃣ 단계 5: Section Images 입력 (7개)

각 섹션마다:
1. 한국어 이미지 업로드 → URL 복사 → 해당 입력란에 붙여넣기
2. 영어 이미지 업로드 → URL 복사 → 해당 입력란에 붙여넣기

### 필수 섹션:
1. **기존방식 (Problem) 이미지**
2. **모션 캡쳐 스튜디오**
3. **포토그래메트리 부스**
4. **모션 콘티 생성 시스템**

### 선택 섹션:
5. 빅데이터 파이프라인 배경
6. AI 분석 알고리즘 배경
7. VRPE 배경

**저장**: "섹션 이미지 저장" 버튼 클릭

---

## 6️⃣ 단계 6: 브라우저 캐시 삭제

1. **Ctrl + Shift + Delete**
2. "전체 기간" 선택
3. "캐시된 이미지 및 파일" 체크
4. 삭제

## ✅ 체크리스트

### Hero Slides (3개):
- [ ] 슬라이드 1: 한국어 이미지 업로드 및 URL 입력
- [ ] 슬라이드 1: 영어 이미지 업로드 및 URL 입력
- [ ] 슬라이드 1: 한국어/영어 제목, 설명 입력
- [ ] 슬라이드 2: 동일하게 입력
- [ ] 슬라이드 3: 동일하게 입력
- [ ] **저장** 버튼 클릭

### Solution Slides (4개):
- [ ] K-POP 댄스: 이미지 + 텍스트 (한국어/영어)
- [ ] 태권도: 이미지 + 텍스트 (한국어/영어)
- [ ] 요가: 이미지 + 텍스트 (한국어/영어)
- [ ] 골프: 이미지 + 텍스트 (한국어/영어)
- [ ] **저장** 버튼 클릭

### Section Images (7개):
- [ ] 각 섹션마다 한국어/영어 이미지 URL 입력
- [ ] **저장** 버튼 클릭

### 최종 확인:
- [ ] 캐시 삭제
- [ ] index.html 확인
- [ ] index_en.html 확인

## 💡 팁

### 같은 이미지를 양쪽 언어에 사용하려면:
동일한 이미지 URL을 한국어/영어 입력란 양쪽에 붙여넣으면 됩니다.

### 이미지 권장 사이즈:
- **Hero Slides**: 1920x1080px (Full HD)
- **Solution Slides**: 800x600px 이상
- **Section Images**: 600x400px 이상

### 이미지 형식:
JPG, PNG, GIF, WEBP (최대 10MB)

---

## 🚨 문제 해결

### 업로드가 안 되는 경우:
- 파일 크기 확인 (10MB 이하)
- 파일 형식 확인 (JPG, PNG, GIF, WEBP)
- 브라우저 콘솔(F12) 에러 확인

### 저장이 안 되는 경우:
- 로그인 상태 확인
- 모든 필수 필드 입력 확인
- 브라우저 콘솔 에러 확인

### 이미지가 안 보이는 경우:
1. URL이 "https://"로 시작하는지 확인
2. 브라우저에서 이미지 URL 직접 접속 테스트
3. 캐시 삭제 (Ctrl + Shift + Delete)
4. 강력 새로고침 (Ctrl + F5)
