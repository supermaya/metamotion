# 관리자 페이지 사용 가이드

## 🎯 처음부터 깔끔하게 시작하기

---

## 1단계: 데이터베이스 초기화 ✅

### phpMyAdmin에서 실행:

1. phpMyAdmin 접속
2. `playelga_metamotion` 데이터베이스 선택
3. **SQL** 탭 클릭
4. 다음 쿼리 복사-붙여넣기:

```sql
TRUNCATE TABLE hero_slides;
TRUNCATE TABLE solution_slides;
TRUNCATE TABLE section_images;
```

5. **실행(Go)** 버튼 클릭

### 또는 파일 사용:
`reset_all_data.sql` 파일 내용 전체를 복사해서 실행

---

## 2단계: 관리자 페이지 접속 🔐

### URL:
- **한국어**: `https://metamotion.io/admin.html`
- **영어**: `https://metamotion.io/admin_en.html`

### 로그인:
기존 관리자 계정으로 로그인하세요.

---

## 3단계: 콘텐츠 입력 📝

### A) Hero Slides (메인 슬라이드) - 3개 고정

각 슬라이드마다 다음을 입력:

**슬라이드 1 예시:**

#### 한국어 (파란색 섹션):
- **제목 (한국어)**: `실감형 VR 교육`
- **설명 (한국어)**: `몰입형 3D 레슨으로 완벽한 동작 학습`
- **이미지 URL (한국어)**:
  1. 하단 "이미지 업로드" 섹션에서 한국어용 이미지 업로드
  2. 생성된 URL 복사
  3. 여기에 붙여넣기

#### 영어 (초록색 섹션):
- **제목 (영어)**: `Immersive VR Education`
- **설명 (영어)**: `Perfect motion learning with immersive 3D lessons`
- **이미지 URL (영어)**:
  1. 영어용 이미지 업로드
  2. URL 복사하여 붙여넣기

**슬라이드 2, 3도 동일하게 입력**

#### 저장:
모든 슬라이드 입력 완료 후 **"Hero Slides 저장"** 버튼 클릭

---

### B) Solution Slides (솔루션 슬라이드) - 개수 제한 없음

추천: 4개 (K-POP 댄스, 태권도, 요가, 골프)

**예시: K-POP 댄스**

#### 한국어:
- **제목**: `K-POP 댄스`
- **설명**: `전문 안무가의 동작을 VR로 배우기`
- **이미지 URL**: (업로드 후 URL 붙여넣기)

#### 영어:
- **제목**: `K-POP Dance`
- **설명**: `Learn choreography from professional dancers in VR`
- **이미지 URL**: (업로드 후 URL 붙여넣기)

#### 저장:
**"Solution Slides 저장"** 버튼 클릭

---

### C) Section Images (섹션 이미지) - 7개 고정

각 섹션마다 한국어/영어 이미지 URL 입력:

1. **기존방식 (Problem) 이미지**
   - 한국어 이미지 URL
   - 영어 이미지 URL

2. **모션 캡쳐 스튜디오**
   - 한국어 이미지 URL
   - 영어 이미지 URL

3. **포토그래메트리 부스**
   - 한국어 이미지 URL
   - 영어 이미지 URL

4. **모션 콘티 생성 시스템**
   - 한국어 이미지 URL
   - 영어 이미지 URL

5. **빅데이터 파이프라인 배경** (선택사항)
6. **AI 분석 알고리즘 배경** (선택사항)
7. **VRPE 배경** (선택사항)

#### 저장:
**"섹션 이미지 저장"** 버튼 클릭

---

## 4단계: 이미지 업로드 방법 📤

### 관리자 페이지 하단의 "이미지 업로드" 섹션 사용:

1. **파일 선택** 버튼 클릭 또는 드래그 앤 드롭
2. 이미지 파일 선택 (JPG, PNG, GIF, WEBP)
3. **"이미지 업로드"** 버튼 클릭
4. 업로드 성공 시 URL이 표시됨
5. **"URL 복사"** 버튼 클릭
6. 해당 입력란에 붙여넣기

### 이미지 권장 사이즈:
- **Hero Slides**: 1920x1080px (Full HD)
- **Solution Slides**: 800x600px
- **Section Images**: 600x400px

---

## 5단계: 저장 및 확인 ✅

### 각 섹션별로:
1. 모든 필드 입력 완료
2. 저장 버튼 클릭
3. 성공 메시지 확인: "○○○가 저장되었습니다!"

### 전체 확인:
1. 브라우저 캐시 삭제: **Ctrl + Shift + Delete**
2. 또는 시크릿 모드: **Ctrl + Shift + N**
3. 페이지 접속:
   - `index.html` → 한국어 콘텐츠 + 한국어 이미지
   - `index_en.html` → 영어 콘텐츠 + 영어 이미지

---

## 💡 팁

### 같은 이미지를 양쪽 언어에 사용하려면:
동일한 이미지 URL을 한국어/영어 입력란 양쪽에 붙여넣으면 됩니다.

### 이미지 미리보기:
이미지 URL을 입력하면 자동으로 미리보기가 표시됩니다.

### 임시로 placeholder 사용:
실제 이미지가 없다면 임시로 다음 사용 가능:
```
https://via.placeholder.com/1920x1080/1e3a8a/FFFFFF?text=Temporary+Image
```

---

## ❌ 문제 해결

### 저장이 안 되는 경우:
- 로그인 상태 확인
- 브라우저 콘솔(F12)에서 에러 확인

### 이미지가 안 보이는 경우:
1. 이미지 URL이 올바른지 확인
2. 브라우저에서 이미지 URL 직접 접속 테스트
3. 캐시 삭제 후 재확인

### API 확인:
```
https://metamotion.io/api.php?type=hero&lang=ko
```
브라우저에서 직접 접속하여 JSON 데이터 확인
